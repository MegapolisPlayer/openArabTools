Types

LightColorD

C-style enum type, no prefix, predefined values for LightColor.
Possible values:

  • LIGHTCOLOR_BLACK
  • LIGHTCOLOR_BLUE
  • LIGHTCOLOR_CYAN
  • LIGHTCOLOR_DARK_GRAY
  • LIGHTCOLOR_GRAY
  • LIGHTCOLOR_GREEN
  • LIGHTCOLOR_CHESSBOARD_BLACK
  • LIGHTCOLOR_CHESSBOARD_DARK
  • LIGHTCOLOR_CHESSBOARD_LIGHT
  • LIGHTCOLOR_CHESSBOARD_WHITE
  • LIGHTCOLOR_LIGHT_GRAY
  • LIGHTCOLOR_MAGENTA
  • LIGHTCOLOR_ORANGE
  • LIGHTCOLOR_PINK
  • LIGHTCOLOR_RED
  • LIGHTCOLOR_TRAFFIC_GREEN_OFF
  • LIGHTCOLOR_TRAFFIC_GREEN_ON,
  • LIGHTCOLOR_TRAFFIC_RED_OFF
  • LIGHTCOLOR_TRAFFIC_RED_ON
  • LIGHTCOLOR_TRAFFIC_YELLOW_OFF,
  • LIGHTCOLOR_TRAFFIC_YELLOW_ON
  • LIGHTCOLOR_WHITE
  • LIGHTCOLOR_YELLOW
Values may not match the original ArabTools colors.

LightColor

Struct type, represents a RGB color.
Methods:

Method Name Method Description
LightColor() noexcept Default constructor
LightColor(const LightColorD aColor) noexcept "Setup" constructor, takes in LightColorD
LightColor (const float aR, const float aG, const float aB) noexcept "Setup" constructor, takes in normalized (0-1) RGB values.
LightColor& Set(const LightColorD aColor) noexcept Sets a color from LightColorD
LightColor& RGB(const float aR, const float aG, const float aB) noexcept Sets a color, takes in normalized (0-1) RGB values.
LightColor& RGB255(const U08 aR, const U08 aG, const U08 aB) noexcept Sets a color, takes in 8-bit RGB values (0-255).
LightColor& Darken() noexcept Darkens color by 25 points, or sets to #000000 (black) if that is not possible.
LightColor& Lighten() noexcept Lightens color by 25 points, or sets to #ffffff (white) if that is not possible.
LightColor& operator=(const LightColor& aColor) noexcept Copy assignment operator
~LightColor() noexcept Default destructor
