Type Name Type Description
OPENARABTOOLS_NO_STANDARD_INCLUDE Define this macro if you have already included all of the necessary headers.
(GLEW, GLFW, iostream, string, vector, thread, chrono, atomic, mutex, numeric)
OPENARABTOOLS_NO_NAMESPACE_OVERRIDE Define this macro to disable the following namespace aliases: ArabTools, oAT, AT
(OpenArabTools is the default namespace and cannot be disabled.)

typedef long double Dec; Decimal number
typedef void(RunConcurrentlyCallback)(const U08, const U64); Type for runConcurrently functions. Args: OpenArabTools Thread Id as U08, Job Id as U64, do not return.
#define NORMAL255(x) x/255.0 Normalizes values in range 0-255 (use for RGB)
#define NORMAL1023(x) x/1023.0 Normalizes values in range 0-1023 (use for RGB)
#define TRAIN auto Trains > cars. Just fact. (this entry might be biased a little bit)
OpenArabTools uses standard C types (float, unsigned int) where applicable.

Types

LightColorD

C-style enum type, no prefix, predefined values for LightColor.
Possible values:

  • LIGHTCOLOR_BLACK
  • LIGHTCOLOR_BLUE
  • LIGHTCOLOR_CYAN
  • LIGHTCOLOR_DARK_GRAY
  • LIGHTCOLOR_GRAY
  • LIGHTCOLOR_GREEN
  • LIGHTCOLOR_CHESSBOARD_BLACK
  • LIGHTCOLOR_CHESSBOARD_DARK
  • LIGHTCOLOR_CHESSBOARD_LIGHT
  • LIGHTCOLOR_CHESSBOARD_WHITE
  • LIGHTCOLOR_LIGHT_GRAY
  • LIGHTCOLOR_MAGENTA
  • LIGHTCOLOR_ORANGE
  • LIGHTCOLOR_PINK
  • LIGHTCOLOR_RED
  • LIGHTCOLOR_TRAFFIC_GREEN_OFF
  • LIGHTCOLOR_TRAFFIC_GREEN_ON,
  • LIGHTCOLOR_TRAFFIC_RED_OFF
  • LIGHTCOLOR_TRAFFIC_RED_ON
  • LIGHTCOLOR_TRAFFIC_YELLOW_OFF,
  • LIGHTCOLOR_TRAFFIC_YELLOW_ON
  • LIGHTCOLOR_WHITE
  • LIGHTCOLOR_YELLOW
Values may not match the original ArabTools colors.

SemaphoreOrientation

C++ style enum-struct type, prefix SemaphoreOrientation::[VALUE]. Represents the orientation of Semaphore.
Possible values:

  • NORMAL
  • UPSIDEUP (=NORMAL)
  • UPSIDEDOWN
  • UPSIDELEFT
  • UPSIDERIGHT
